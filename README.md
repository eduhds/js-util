## Instalação

```sh
npm i eduhds/js-util
yarn add eduhds/js-util
pnpm add eduhds/js-util
```

## Desenvolvimento

```sh
# Limpar outputs
npm run clean

# Compilar lib
npm run tsc

# Executar testes
npm run test

# Gerar documentação
npm run docs
```
